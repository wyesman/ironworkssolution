# Iron Works Solution - Premium Lead Generation Website

A high-performance, conversion-optimized website for a Southern California iron fencing and metal fabrication company. Built with Next.js 15, TypeScript, and Tailwind CSS.

## üé® Design Philosophy

Inspired by Compass Ironworks, this website combines elegant craftsmanship with modern lead-generation tools. The design features:

- **Color Palette**: Deep charcoal (#1a1a1a), warm gray (#3c3c3c), cardinal red accent (#C41E3A), cream (#f8f6f3), metallic silver
- **Typography**: Playfair Display (headings) + Open Sans (body)
- **Visual Style**: Premium, confident, craftsmen-built aesthetic with brushed metal textures and ornamental scrollwork

## üé® Customization

### Colors
Update colors in `src/app/globals.css` under the `:root` and `.dark` selectors.

### Components
All UI components are in `src/components/ui/` and can be customized.

### Content
Update content directly in component files:
- Services: `src/components/Services.tsx`
- Testimonials: `src/components/Testimonials.tsx`
- Blog posts: `src/components/Blog.tsx`
- Gallery items: `src/components/Gallery.tsx`
